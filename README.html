공공데이터 지도 시각화 사이트 – 주간 개발 계획

사용스택: vue / javascript / mysql / naverMap_api /

목적: 3년간의 실무 경험(데이터 연동, 지도 시각화)을 포트폴리오 사이트로 보여주기

기능: 데이터 적재(공공데이터 포털), 네이버지도 마커 표시, 상세 팝업 및 차트/이미지 시각화

✅ Day 1 (월요일) – 기획 및 구조 설계

  • 데이터 선정 및 공공 API 검토
  • 데이터베이스 스키마 설계
  • 간단한 적재 스크립트 작성
  • 네이버 지도 API 기본 세팅
  • 기술 스택 및 폴더 구조 확정

✅ Day 2 (화요일) – 데이터 연동 및 지도 시각화

  • API 연동 후 데이터 적재 기능 완성
  • 지도 띄우기 + 테스트 마커 표시
  • DB에서 마커 정보 동적으로 불러오기
  • 마커 클릭 시 팝업 기본 구현

✅ Day 3 (수요일) – 팝업 및 시각화 연동

  • 팝업에 상세 정보 출력
  • 차트 시각화 라이브러리 연동
  • 추가 이미지 등 콘텐츠 연동

✅ Day 4 (목요일) – UI 개선 및 예외 처리

  • 레이아웃 정리 (반응형 포함)
  • 로딩/에러 등 사용자 경험 개선
  • 데이터 누락 대비 예외 처리
  • 데이터 적재 자동화 스크립트 정리

✅ Day 5 (금요일) – 배포 및 마무리

  • 사이트 소개 글 작성 및 UI 삽입
  • 배포 (Vercel, Netlify, 서버 등)
  • 포트폴리오용 캡처, README 정리
  • 최종 점검 및 백업
